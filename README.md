🍕 Pizza Sales Analysis Project
📌 Overview

This project focuses on analyzing pizza sales data to extract meaningful insights and improve business decision-making. The analysis includes data cleaning, transformation, and visualization to identify trends and performance metrics.

🎯 Objectives
Analyze overall sales performance
Identify top and low-performing pizzas
Understand customer ordering patterns
Track revenue trends over time
Provide actionable business insights

🛠️ Tools & Technologies
Microsoft Excel
SQL (for data querying)
Power BI (for dashboard & visualization)

📊 Key Features
Interactive dashboard
KPI tracking (Revenue, Orders, Quantity Sold)
Time-based analysis (Daily, Monthly trends)
Category-wise performance (Veg/Non-Veg, Size, Type)
Top & bottom-selling product analysis

📈 Key Insights
Peak sales hours and busiest days identified
Best-selling pizza categories drive major revenue
Seasonal trends affect order volume
Certain pizzas consistently underperform

🚀 Business Impact

This project helps businesses:

Improve product strategy
Optimize inventory
Increase revenue through data-driven decisions
